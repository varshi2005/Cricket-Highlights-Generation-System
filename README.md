## Overview

This project automatically converts a full cricket match into a 10-minute highlight video using embedded systems, networking, and intelligent event detection.

---

## Key Components

- Embedded cameras and microphones  
- Cloud-based processing  
- AI-based event detection  
- Networking protocols (TCP, UDP, HTTP)  

---

## Highlight Events

- Wickets  
- Boundaries (4s, 6s)  
- Milestones  
- Match-winning moments  

---

## Technologies Used

- Networking (TCP/UDP, RTSP)  
- Cloud Computing  
- Video Processing  
- AI Concepts
